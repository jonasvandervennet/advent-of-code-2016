# Advent Of Code 2016

A Rust implementation of the 2016 edition of [Advent of Code](https://adventofcode.com/2016/).

## Build

```sh
cargo build
```

## Test

```sh
cargo test
```

## Run

### Run all

```sh
cargo run
```

### Run a specific day

```sh
cargo run -- --day 1
```
